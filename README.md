# BLDC FOC Power Stage Prototype

## Overview

Custom three-phase power stage designed for BLDC motor control applications.

The project includes a complete power stage with gate drivers, MOSFET half-bridges, current sensing circuits and back-EMF measurement inputs intended for future Field Oriented Control (FOC) implementation.

## Features

- Three-phase inverter topology
- MOSFET half-bridges
- IR2104 gate drivers
- Current sensing inputs
- Back-EMF measurement circuits
- External control signal inputs
- PCB designed in KiCad

## Hardware

- IR2104 gate drivers
- N-channel MOSFETs
- Current shunt measurement
- Back-EMF sensing
- Bootstrap supply circuits
